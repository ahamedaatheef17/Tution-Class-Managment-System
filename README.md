# Tuition Class Management System (TCMS)

This project is developed as an academic assignment using the Waterfall model.

## Problem Statement
Many tuition classes manage student records, attendance, and fees manually, which leads to errors and data loss.

## Objectives
- Maintain student records
- Manage class schedules
- Track attendance
- Handle fee payments
- Generate basic reports

## Scope
- Student management
- Attendance management
- Fee tracking
- Report generation

## Out of Scope
- Online payment systems
- Mobile applications
- Advanced analytics
Academic project for Tution Class Management System using the waterfall Model
